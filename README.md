# To-Do App (Spring Boot)

Простой менеджер задач (To-Do приложение), реализованное с использованием Java и Spring Boot.

## Технологии

- Java 11+
- Spring Boot
- PostgreSQL
- Spring Data JPA
- Lombok

## Запуск проекта

1. Установить PostgreSQL и создать базу данных (например: `todo_db`)
2. Настроить файл `application.yml` с параметрами подключения
3. Выполнить сборку и запуск приложения:

```bash
mvn clean install
mvn spring-boot:run
